#Tean Tour 2020

Early in November, Atlassian dropped a bunch of new product announcments. Most of these seem Cloud only, for the foreseeable future. Here are my stream of conciousnes notes while I watch it all.

Trello
Lots of talks on Trello. I haven't really used Trello professionally. I still feel like it's a bit Jira-lite. Atlassian does seem to be adding more features to it to make it applicable for more teams, but it still feels like a competing, not complimentary, proudt to Jira. I don't see a lot of companies that would be using both, so I don't think managing Trello will come up ofter in my professional career, but who knows.

New Jira Core
For as long as I have worked in Jira, Jira Core was the base product, and then companies added Jira Software or Jira Service Desk on top of that. I've literally never seen an organization that was only using Core. Atlassian is now breaking Core off into it's own product for "business teams." Details on what that actually will include are sparse, but I got the impression it will be aimed towards non-technical business teams that need good task management, with an entire new interface. How this will integreate with Jira Software or Jira Service Management, I'm not sure yet. Same with the pricing. I could see the appeal for some teams though. In the past whenever I've set up Jira for a business project, I usually spun up a Software project with a Kanban board, and that works, but something that's more tailored to business teams could be nice. We'll have to wait and see what extra functionality it brings though. The most up-to-date news on this new product should be found here https://www.atlassian.com/software/jira/core

Confluence Cloud
A new editor is coming, along with a bunch of new tools and templates. Previously I've found the Confluence templates to be fairly underwhelming, I've never really used them, but some of the new ones look pretty slick. A lot more support for embedding information into a Confluence page, everything from a file hosted on Dropbox to embedding a Twitter feed. Oh and now it has page archiving, finally. I really hope that last one comes to On-Prem eventually.

Atlassian Status
The video on what exactly this is wasn't that clear to me. It seems to be a product that makes it easier for Project Leads to provide high level information and status reports about the projects they manage. Which is always nice. But from the video I couldn't quite tell if this was an entirely stand-alone thing, or something that lives within Jira or Confluence. There's a website for this new product up, though at the time of this writing it's just an email signup link for a beta invite. https://www.atlassian.com/software/watermelon
